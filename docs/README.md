# Azure OpenAI 入門ハンズオンラボ
このハンズオンラボでは、以下のことをカバーします。

- このハンズオンラボを始めるまえに必要な環境設定
- Azure OpenAI を使うための初期設定
- Azure OpenAI の基本的な機能を試してみる
- Azure OpenAI に自前のドキュメントを組み込む
- Azure OpenAI を使った賢いアプリケーションを実装する
